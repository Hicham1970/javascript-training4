the BOM object : the navigator

The navigator ? c'est quoi? the navbar? le navigator est le bateau qui nous permet de naviguer sur le web . The navigator object is a property of the window object. Chaque browser ou navigateur possède son propre système de navigation , si j'ose dire, qui lui est propre. On peut acceder a cet objet par la variable window.navigator. Et on utiliser cette propriété pour savoir si le navigateur a activé les cookies ou non.
