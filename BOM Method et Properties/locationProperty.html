Location property

Location = emplacement, href = hypertext reference adresse url,

Location est une creature du window , un objet de window et le href une propriété de la location object entre autres.

Est ce qu'on peut la changer? oui avec get est set

Ce que la location.href accept 1. adresse url absolut 2. adresse d'une page a l’intérieur de la page courante 3. accepte aussi un hash id dans la même page 4. accepte aussi un ftp ou file transfert protocol , accept aussi l'envoie d'email (location.href="mailto:h.gmail.com")
