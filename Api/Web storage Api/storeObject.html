Storing a Javascript Object

Il faut savoir que toutes les données enregistres dans local storage sont dans une forme de string,

Si on veut emmagasiner un objet javascript dans le local storage, on doit le convertir en string , cette opération s’appelle la sérialisation

On l'ajoute apres l'avoir convertit en string par la method stringify de Json.

On peut le récupérer avec la method parse de Json.
