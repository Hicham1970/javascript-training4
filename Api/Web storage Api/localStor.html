The localStorage Window Object

Dans cet objet on veut stocker nos données pour les récupérés quand on le veut, par exemple un choix de couleurs qu'on veut utiliser ultérieurement

On peut ajouter un local storage avec la method window.localStorage.setItem("key","value")

SetItems()

Une autre manière de faire ça : window.localStorage.KeyName = "Value" , directement . Il y a une autre façon : window.localStorage["font"] = "Helvetica" ; par exemple.

Dans le monde réel , si un utilisateur choisit une couleur ou un type de font, son choix est enregistré dans le local storage , et si on fait un rafraîchi de la page ou un redémarrage de la page, le browser va afficher les couleurs choisit et les traits de font de l'utilisateur

GetItems()

Maintenant qu'on a créer notre localStorage, Comment l'utiliser?? Avec cette sentence :window.localStorage.getItem('keyName') ou encore window.localStorage.keyName, et on peut même écrire window.localStorage["keyName"];

Et pour utiliser cette data dans notre page web la couleur du body sera celle créer dans le local storage cad la couleur rouge #F00

RemoveItem()

On a ajouter une data avec setItem("keyName", "Value");puis on appelé cette donnée avec la method getItem("keyName').

Il y'a la méthode LocalStorage.key(0) qui nous affiche le premier item de la data

On supprime une donnée du localStorage avec la sentence : window.localStorage.remove('keyName').
