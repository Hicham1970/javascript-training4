Session Storage Object

La session storage est identique au local storage sauf que ce dernier reste dans la machine si on ferme la session et la session storage est effacé apres fermeture de la page.

Il est preferable de travailler avec la session storage si on veut preserver la confidentialité des données de l'utilisateur

Pour ajouter des données dans la session storage, on utilise la méthode setItem()

Pour récupérer des données de la session storage, on utilise la méthode getItem()

Supposons que nous avons un input ou l'utilisateur va entrer son nom





A chaque fois que l'utilisateur entre un nom, il sera enregistré dans la session storage.
