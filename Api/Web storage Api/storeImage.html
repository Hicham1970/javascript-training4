Comment emmagasiner une photo dans le localStorage




Vous savez bien que le local storage et la session storage n'acceptent les data que sous forme de string , alors notre fichier .jpg devrais être convertit en string ou plus précisément en data url avec la creation d'une nouvelle instance de FileReader() .

Le résultat obtenu après la conversion de la donnée en string, on peut l'emmagasiner dans le local ou la session Storage

Maintenant que la photo est enregistrée dans notre local storage, on peut l’appeler pour l'afficher

On va installer un écouteur d'évènement sur le document au moment du chargement de la page, on va récupérer l'image emmagasinée, si elle existe bien sur, et puis la donner a l'element image comme source
