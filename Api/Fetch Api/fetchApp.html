An Other Example

The fetch() est une fonction intrinsèque de javascript qui permet de faire des requêtes HTTP pour récupérer des resources.

Ces resources peuvent être des images, des fichiers ou des données sous format Json



Javascript est reconnu par son fil d’exécution asynchrone , cette méthode vient pour combler cette lacune; On l'utilise pour interagir avec les apis, pour récupérer et envoyer les données d'une façon asynchrone vers le web server

Cette fonction accepte 2 arguments: l'url et {options}
