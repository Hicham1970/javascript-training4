Set a cookie

In the example to follow, we will create a cookie that stores the name of a visitor.

The first time a visitor arrives to the web page, he/she will be asked to fill in his/her name. The name is then stored in a cookie.

The next time the visitor arrives at the same page, he/she will get a welcome message. For the example we will create 3 JavaScript functions:

The first function to set a cookie value, the second function to get a cookie value and the third function to check a cookie value
